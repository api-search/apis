# API Search - APIs

This repository stores individual API records for the [APIs.io](https://apis.io) search network.

## About

Each API record represents a single API indexed by the network, including its metadata, endpoints, and documentation references. Records are generated from [api-evangelist](https://github.com/api-evangelist) repos and used to build the `_apis/` Jekyll collection on the APIs.io network site.

## Related Repositories
