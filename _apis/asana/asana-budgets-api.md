description: The Asana Budgets API allows developers to manage budget resources for projects. A budget object represents a budget for a specific parent resource such as a project and tracks values in either time or cost formats. When the type is time, values are expressed in minutes; for cost, values use the domain's currency. The API supports creating, retrieving, updating, and deleting budgets.
