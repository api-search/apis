description: The Asana Project Statuses API allows developers to manage progress updates on projects. A project status includes descriptive text and color codes indicating the project state such as green for on track, yellow for at risk, red for behind, or blue for on hold. The API supports creating, retrieving, and deleting project statuses. Note that project statuses are deprecated for new integrations in favor of the Status Updates API.
